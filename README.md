# Mash-Up Android 세미나 아카이브

Mash-Up Android팀의 10분 세미나 자료를 기수별로 아카이빙합니다.

<br>

## 세미나 자료 등록 방법

1. 상단의 **Issues** 탭으로 이동
2. **New Issue** 클릭 후 `세미나 자료 등록` 선택
3. 기수, 이름, 주제, 자료 입력 후 제출

> 자료는 PDF 파일, 블로그 링크, 노션 링크 등 어떤 형식으로도 등록 가능합니다.

자세한 내용은 [등록 가이드](./INSTRUCTION.md)를 참고해주세요.

<br>

저장소 운영 및 관리 방법은 [관리자 가이드](./ADMIN.md)를 참고해주세요.
