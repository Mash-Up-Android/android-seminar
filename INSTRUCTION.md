# 세미나 자료 등록 방법

이 문서는 Mash-Up Android팀 팀원들을 위한 세미나 자료 등록 가이드입니다.

세미나 자료는 횟수 제한 없이 등록할 수 있습니다. 세미나를 진행할 때마다 이슈를 새로 등록해주세요.

<br>

## 등록 절차

**1. Issues 탭으로 이동**

상단 메뉴에서 **Issues** 탭을 클릭합니다.

**2. New Issue 클릭**

우측 상단의 **New Issue** 버튼을 클릭합니다.

**3. `세미나 자료 등록` 템플릿 선택**

템플릿 목록에서 `세미나 자료 등록`의 **Get started** 버튼을 클릭합니다.

**4. 내용 입력 후 제출**

아래 항목을 입력하고 **Submit new issue**를 클릭합니다.

| 항목 | 설명 |
|------|------|
| 기수 | 본인의 활동 기수를 선택합니다 |
| 이름 | 본인의 이름을 입력합니다 |
| 주제 | 발표한 세미나 주제를 입력합니다 |
| 자료 | 블로그 링크, 노션 링크, PDF 파일 등을 등록합니다 |

<br>

## 자료 등록 형식

자료는 **어떤 형식이든** 등록 가능합니다.

- **블로그 링크** : URL을 그대로 붙여넣기
  ```
  https://medium.com/my-post
  ```
- **PDF 파일** : 자료 입력창에 파일을 드래그앤드롭하면 GitHub 링크로 자동 변환됩니다
- **노션, 구글 슬라이드 등** : 공유 URL을 그대로 붙여넣기

<br>

## 등록 후 처리

이슈를 제출하면 자동으로 아래 과정이 진행됩니다.

```
이슈 제출
   ↓
자동으로 해당 기수 README에 자료 추가
   ↓
등록 완료 댓글 작성
   ↓
이슈 자동 Close
```

별도로 추가 작업을 할 필요가 없습니다.

<br>

## 잘못 등록한 경우

잘못 등록된 자료는 아래 절차로 직접 삭제할 수 있습니다.

삭제는 **이슈 단위**로 처리됩니다. 여러 세미나를 등록한 경우, 삭제하려는 세미나에 해당하는 이슈로 이동해서 진행해주세요.

**1.** 삭제하려는 세미나의 이슈(Close된 상태)로 이동합니다.

**2.** 우측 **Labels** 항목에서 `delete-seminar` 라벨을 추가합니다.

**3.** 자동으로 아래 과정이 진행됩니다.

```
delete-seminar 라벨 부착
   ↓
해당 기수 README에서 자료 행 제거
   ↓
삭제 완료 댓글 작성
```

> **주의:** 삭제 시 이슈에 입력한 이름과 주제를 기준으로 아카이브에서 해당 항목을 찾아 제거합니다.
